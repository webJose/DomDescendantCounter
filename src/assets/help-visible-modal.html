What counts as "Visible"?

An element is counted as visible if it meets all of the following criteria:

  • Display: The element's computed style display is not none
  • Visibility: The element's computed style visibility is visible
  • Opacity: The element's computed style opacity is not 0
  • Dimensions: The element has non-zero width and height (via getBoundingClientRect())
  • Viewport intersection: The element's bounding rectangle intersects with the current viewport

Note: Elements that are positioned completely off-screen or outside the viewport will NOT be counted as visible, even if they meet the other criteria.

This definition helps distinguish between elements that are actually rendered and visible to users versus those that are hidden through CSS or positioned outside the visible area.
